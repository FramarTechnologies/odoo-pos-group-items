# POS Group Items

**Sell product kits as one item in Odoo POS while tracking stock accurately for all components.**

---

## Description

POS Group Items allows you to sell bundled products, kits, or packages as a single item in Odoo POS.  
When a kit is sold, each component is automatically posted and deducted in the backend, ensuring accurate inventory, POS session tracking, and reporting.

---

## Key Features

- Sell kits or bundled products as a single item in POS  
- Automatically deduct stock for each component  
- POS sessions and inventory reports remain accurate  
- Simple configuration directly from the Product form  
- Seamless integration with Inventory, POS, and Accounting  

---

## Example Use Case

**Total Gas 6KG Full Kit**  
- 6KG Gas Cylinder  
- Regulator  
- Gas Pipe  

When sold in POS, Odoo automatically posts all three items individually in the backend, keeping stock counts correct while showing a single kit on the POS screen.

---

## How to Use

1. Create a product and mark it as **POS Group Item**  
2. Add component products and their quantities in the **Product Group** section  
3. Sell the kit in POS as one product  
4. Odoo will automatically explode the kit into its components for stock tracking
